SRC高级功能

作为一个抽取学生姓名的程序，以下功能显得不必要。这是为了希望在seewo上可以使用Python的一些功能而作的拓展

mode> eval

使用 print(eval(input('eval> '))) 处理，解析Python表达式，用于进行一些数学运算和单行Python代码注入

mode> exec

接受多行输入，并作为Python代码运行。您可查看源代码以临时自定义功能

mode> run

将文件作为Python文件读取执行，运行一些简单的Python代码，此功能为试行，未知其可靠性

接受参数

SRC可以接受参数，并将最后一个参数当作文件名，尝试将此文件当作Python文件运行

SRC再编译

SRC是使用python3.12.4编写及此版本下的pyinstaller编译而成，SRC和SRC-GUI的源代码是完全一样的， 他们只有优先运行文件的区别。编译random_choice.py即为SRC，编译gui.py即为SRC-GUI

pyinstaller -F -n SRC src/SRC/random_choice.py
